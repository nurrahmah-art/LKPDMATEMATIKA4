Aplikasi Matematika Kelas 4 - SDIT Ar-Rahmah

Cara membuka:
1. Buka file index.html menggunakan Google Chrome, Microsoft Edge, atau browser lain.
2. Gunakan menu Dashboard, Materi, Simulator, Kuis, LKPD, Rubrik, dan Mode Guru.
3. Tekan tombol Cetak LKPD/Rubrik jika ingin mencetak lembar kerja atau rubrik.

Isi aplikasi:
- ATP Matematika Kelas 4 / Fase B.
- Materi pembagian bilangan cacah sampai 9.999.
- Simulator bagi panjang.
- Latihan cepat dengan pengecekan otomatis.
- Asesmen awal 5 soal sesuai lampiran perangkat ajar.
- LKPD digital berdasarkan kasus Pak Amir.
- Rubrik penilaian sikap, pengetahuan, dan keterampilan.

Catatan:
Aplikasi ini berjalan offline dan tidak membutuhkan database.
